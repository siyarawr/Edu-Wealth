# Icon Requirements for Chrome Web Store

You need to create PNG icons in these sizes:
- icon16.png (16x16 pixels) - Toolbar icon
- icon48.png (48x48 pixels) - Extensions page
- icon128.png (128x128 pixels) - Chrome Web Store listing

## Design Guidelines
- Use a simple, recognizable design
- Ensure good visibility at small sizes
- Use the Edu Wealth teal/cyan color (#14b8a6)
- PNG format with transparency

## Quick Creation Options
1. Use a design tool like Figma, Canva, or Photoshop
2. Create an "EW" logo with the teal gradient background
3. Export at all three sizes

For testing, you can use any placeholder images renamed to these sizes.
